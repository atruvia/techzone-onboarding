# Notizen

- **Zielgruppe:** Atruvia-Entwickler*innen, neu in BMAD. Die Umgebung ist vorbereitet —
  keine Installationsschritte zeigen (`npx bmad-method install` bewusst weggelassen).
- **Sprache:** Deutsch, »Wir«/»du«, Gender-Stern. Englisch nur für BMAD-Eigennamen
  (PM, Architect, Scrum Master, Dev, PRD) und Befehle (`bmad-*`).
- **Frozen, aber resumable:** Dieser Lernbereich ist als statische HTML für GitHub Pages
  eingefroren, bleibt aber ein voller teach-Workspace. Wer den Ordner in der Claude CLI
  öffnet, kann die Session live fortsetzen.
- **Versionsstand:** BMAD-Befehle und Phasen zitiert nach docs.bmad-method.org (Stand
  Juni 2026). Bei einem Versionssprung gegen die Doku gegenprüfen — die `bmad-*`-Namen
  können sich ändern.
- **Design:** Lektionen/Reference bauen auf der teach-`lessons.css`; das Onboarding fügt
  nur `assets/page.css` hinzu (Back-to-Hub-Leiste + Lektionsliste + Cheat-Sheet-Tabelle).
